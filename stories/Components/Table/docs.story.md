# Table

The `Table` component is a versatile and customizable table that supports various features such as sorting, filtering, and pagination. It provides an easy-to-use interface for displaying data in a structured and organized manner. The `Table` component is designed with accessibility and responsiveness in mind, ensuring a seamless user experience. Whether you need a simple table or a more complex one with advanced features, this component has you covered.

## Installation

```bash
npx @lenne.tech/cli components add Table
```

```bash
npx @lenne.tech/cli components add TableDraggable
```
