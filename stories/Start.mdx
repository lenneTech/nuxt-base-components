# Nuxt Base Components

The lenne.Tech Nuxt Base Components is a library for your Nuxt 3 environment

## Including

- [Tailwind](https://tailwindcss.com/)
- [Pinia](https://pinia.vuejs.org/)
- [VueUse](https://vueuse.org/)
- [Nuxt Base](https://github.com/nuxtbase)
- [EsLint](https://eslint.org/)
- [Cypress with Cucumber, Unit Tests](https://www.cypress.io/)
